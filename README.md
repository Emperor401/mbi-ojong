# Mbi Ojong — AML Investigation Analyst Portfolio

Personal portfolio and consulting site for **Jiles Agbor Mbi Ojong**, an AML Investigation Analyst specialising in consumer transaction monitoring, financial crime investigation, and compliance risk assessment.

## What's here

- **Home** — introduction, career milestones, and services overview
- **About** — background, expertise, and mission
- **Services** — AML investigation, SAR reporting, KYC/CDD, and risk assessment offerings
- **Certificates** — professional certifications and credentials
- **Contact** — consultation request form with email delivery

## Tech stack

- [Next.js](https://nextjs.org) (App Router)
- [Tailwind CSS](https://tailwindcss.com)
- [GSAP](https://gsap.com) for animation
- [Nodemailer](https://nodemailer.com) for contact form
